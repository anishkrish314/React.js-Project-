PopCart - Frontend for E-commerce Website

This is a simple e-commerce website using React.js and CSS. This is a small online store that features some products and a cart.

Features
Displays products with images and names
Allows products to be added to a cart or removed from the cart
Has simple cart functionality that keeps correct reference to product id and name
Included login and Signup page
Responsive design for desktop screens

Challenges I Faced and How I Overcame
I had to fix errors arising from imports, by confirming the spelling was correct and the correct path was imported
For images not being aligned, I had to check that the correct image number matched the products
For the cart issues, I had to check the product id and name

Technologies Used
React.js
CSS

Note
This is a frontend project only, and does not have any backend or database connected
